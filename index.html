Common CSS Stylings

By James C.H. Lim

Common Properties

Property Used by Description
font-weight strings of class, id or attribute Takes a value of 1-1000, or 'bold', 'bolder', 'lighter', 'normal', 'initial' or 'inherit'. Changes the thickness of the font line.
font-family strings of class, id or attribute Changes the font character set to any font installed on the computer, or if linked to a google font, any font available on their repository.
text-size strings of class, id or attribute Controls the size of the text, accepting normal size variables in pixels, rem and em.
color and background-color Colors: strings of class, id or attribute
background-color: applies to all enclosed element 		color changes the color of the font to accepted values of hex, #ffffff, 147 named specified colors, eg. black, azure, limegreen or an rgb function, eg. rgb(256,256,256) or rgba(256,256,256,0.5), where the final value of rgba controls opacity.
text-align strings of class, id or attribute Aligns the text in child elements of a enclosed element to left, center or right of the parent element.
padding and margin All child elements Padding seperates the content of the element to the space the element takes up, this is useful when using borders or a background images to easily seperate text from its surroundings.

Margins on the other hand, seperates the space the element takes up from surrounding elements. This distances can be used to distinguish an element from surrounding elements.
display All child elements Changes the way that elements are orders on the page. Default display is block, which makes each element take up its own row on the page. The other options are inline, which makes each element try to be inline when possible, and inline-block, a version of inline which allows greater flexibility and respect of margins and paddings.


Common Values

Value Used by Description
1rem, 16px, 1em Padding and margin, text-size 1rem is equivalent to 16 pixels, and is more easily adaptable to the changing fonts of a browser. To be more specific, one can use pixels (px). Finally, there is em, which sizes based on the default size of the parent element
black, #123abc, rgb(256,256,256) color and background-color There are value examples which color and background-color can parse. 147 named values found here, and a rgb function which is equivalent to hex. A new function that has emerged is rgba, which has a fourth value of 0-1 which controls the opacity.
inline, block, inline-block display Changes the display type of elements. Block means each element occupies it's own space while inline means elements will fit inline horizontally when possible. Inline-block is similar to inline except it allows the setting of margins and paddings, and respects height and width.
left, right, center text-align Aligns text of a child element in proportion to its parent element or its default settings.
